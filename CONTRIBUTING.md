# Contributing Guidelines

(WIP)

## Run the project

Install deps:

```
yarn
```

Run the tests:

```
pnpm run test:unit --watch
```

Build the library in dev mode:

```
pnpm run dev
```

Build the library for production:

```
pnpm run build
```

### Demo

Run the demo:

```
cd demo-src
yarn
pnpm run dev
```

Build the demo:

```
pnpm run build
```
